\section{Einführung}\label{label_introduction}
\pagenumbering{arabic}
\setcounter{page}{1}

Der Hauptteil dieses Dokuments beschreibt, wie QGIS~\CURRENT aus dem Quellcode
erstellt werden kann. Die Instruktionen sind für GNU/Linux und andere POSIX
Systeme, die eine entsprechende BUILD-Umgebung bereitstellen. Wenn Sie unter
FreeBSD arbeiten, vergleichen Sie bitte 
\url{http://qgis.org/index.php?option=com_content&task=view&id=84&Itemid=86} für
weitere Tipps und Informationen.

Die Installation unter MS Windows und Mac OS X ist sehr einfach und wird im
unteren Abschnitt beschrieben.

Sie müssen nicht alle für QGIS notwendige Software aus dem Quellen erstellen und
installieren. Wenn Ihr Betriebssystem eine akzeptable Version der entsprechenden
Software bereitstellt, können Sie diese vor dem Erstellen von QGIS installieren.
Vergewissern Sie sich, dass Sie die Entwicklerpakete mit installiert haben,
falls diese als Extrapakete zur Verfügung stehen. QGIS benötigt die
'Header'-Dateien.

Die aktuelleste Version dieses Dokuments können Sie finden unter der URL:  
\url{http://qgis.org/index.php?option=com_content&task=view&id=106&Itemid=89}.
