\section{SQL-Anywhere Plugin}\label{sec:sqlanywhere}

% when the revision of a section has been finalized, 
% comment out the following line:
% \updatedisclaimer

SQL Anywhere ist ein proprietäres relationales Datenbank-Managementsystem (RDBMS)
von Sybase. SQL Anywhere umfasst Unterstützung für räumliche Daten einschließlich OGC, 
Shape-Dateien etc. sowie eingebaute Funktionen für den Export nach KML, GML und ins 
SVG-Format.

The \toolbtntwo{sqlanywhere}{SQL Anywhere} Plugin provides a native data provider 
added to QGIS under the GPL v3. The Plugin allows to connect to this SQL 
Anywhere. The \dialog{Add SQL Anywhere layer} dialog is similar in functionality 
to the dialogs for PostGIS and SpatiaLite.

Für die \toolbtntwo{sqlanywhere}{SQL-Anywhere} Erweiterung wurde ein nativer 
Daten-Provider unter GPL v3 Lizens hinzugefügt. Das Plugin erlaubt damit die Anbindung. 
Der \dialog{SQL-Anywhere Layer hinzufügen} Dialog ist in der Funktionalität ähnlich wie 
die Dialoge für PostGIS und SpatiaLite.
