\minisec{Instalar el complemento de Algebra Raster}

Para usar las funcionalidades en QGIS, debe instalar complemento de python
de Algebra Raster con el instalador \mainmenuopt{Obtener Complementos de Python}
(vea la sección \ref{sec:load_external_plugin}). Entonces selecciónelo y carguelo 
con el Administrador de Complementos. Para eso haga clic en el menú \mainmenuopt{Complementos} > 
\mainmenuopt{Administrar complementos}, seleccionar \dropmenuopt{Algebra Raster} y clic 
\button{OK}. Un nuevo ícono \toolbtntwo{copyright_label}{Algebra Raster} aparecerá en la barra de herramientas.
